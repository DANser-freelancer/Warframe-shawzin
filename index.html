Shawzin music compiler

Hi, this app will translate musical notes into shawzin code that you can paste in your favourite sawzin and play.

Rules:

  • Shawzin has a concrete set of notes for each scale, you must select your musical scale accordingly.
  • If the note you want is not present in this scale you must choose the closest note or skip it
    (try to correlate the highest notes and work off of that).
  • The structure of the notes is this: (1-4096)+(L/M/H)+(C/D/E/F/G/A/B)+[S]("[]" means optional).
  • (L/M/H) stands for low/mid/high, [S] stands for sharp.
  • To play a combination of notes in the same moment you must write them using '+'
    with only one timing indication (25lg+mds,55mf...) and in ascending order,
    only maximum of 3 notes can be played at the same time, not all notes can be played together.
  • Each note must be separated by a comma (2LD, 34HCs,...).
  • You must add number (1-4096) before each note to indicate it's position on the timeline, the first note must start at position 1.
  • Shawzin has 4 16th of a second per beat 4 beats per measure, in other words you can play 16 consecutive notes in one second,
    max length of the track is therefore 8min 32sec.
    Note: these are really small notes so if your music seems to run too fast try enabling slow mode in your shawzin.
  • The input field is case insensitive, you can write "lD" or "Ld" or "ld".
