Shawzin music compiler

Hi, this app will translate musical notes into shawzin code that you can copy&paste to play your favourite shawzin.
Made by DANser#6915 (discord).

Rules:

  • Shawzin has a concrete set of notes for each scale, you cannot transition over to the other scale mid-play as you would do with a piano for example.
  • Try to correlate the highest notes and work with that, use the "Transpose notes" feature, click on the field to transpose, line 2 is where you would find the clef.
  • The structure of the notes is this: (1-4096)+(A/B/C/D/E/F/G/H/I/J/K/L).
  • Each note must be separated by a comma (2A, 34F,...).
  • You must add number (1-4096) before each note to indicate it's position on the timeline, you can start from any position.
  • To play a combination of notes in the same moment you must write them using '+' with only one timing indication, and with no repeating notes.
    Ex. (25A+B+C, ...)✔️|||(25C+A+B, ...)✔️|||(25C+A+B+B, ...)❌|||(25A,25B,25C, ...)❌|||(25A+25B+25C, ...)❌.
    Unfortunately not all notes can be played together, more info here. *now more notes can be played together, working on a feature for near full combination of notes.
  • Shawzin can only play 1/16 notes with 4/4 musical time, in other words you can play 16 consecutive notes in one second, max length of the track is therefore 8min 32sec. If you will only need 1/4 or 1/8 notes you can change all or neither of them using the "Choose note duration" feature, this will essentially add space between your notes as if they were longer. These are really small notes so if your music seems to run too fast try enabling slow mode in your shawzin.
  • The input field is case insensitive, you can write "A" or "a". Also you can use spaces and newlines, and you can resize it.
  • The database shows name of the song + name of the band + approximate length of the track*(It may vary if you enable slow mode). When searching for a song - write song/band name and click on the field to search.
  • Check this Shawzin documentation to see the note relations.
  • Click on the logo (top of the page) to join discord server for Warframe musical stuff.
